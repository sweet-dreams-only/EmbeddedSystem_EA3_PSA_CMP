1. Installation
===============

This installation copies the current version of the 
osCAN TimingAnalyzer onto the local disk.


2. Examples
===========

A hardware independent example is included in the installation.
It can be found in the examples\ subdirectory.


3. Documentation
================

The installation contains all documentation in electronically 
readable format. The file UserManual_osCAN_TimingAnalyzer.pdf
can be found in the doc\ subdirectory and is accessible via
the Windows Start Menu.


4. Im/Exporting attributes from/to OIL files
============================================

osCAN TimingAnalyzer can read the attributes (Deadline,
Period, etc.) provided in OIL files. Currently it is not 
possible to export these attributes back to OIL files.
